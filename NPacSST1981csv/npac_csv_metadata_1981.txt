  Description:
    Abstract: The Sea Surface Temperature (SST) data of the nearshore region of the North Pacific show temperature ranges in degrees C using points whose locations correspond to the centroids of AVHRR Pathfinder version 5 monthly, global, 4 km data set (PFSST V50). The pathfinder rasters are available from the Physical Oceanography Distributed Active Archive Center (PO.DAAC), hosted by NASA JPL.  The data points in this dataset lie within a 20 km buffer from the GSHHS (Global Self-consistent, Hierarchical, High-resolution Shoreline) coastline. The GSHHS vector data are available from the National Geophysical Data Center (NGDC). Furthermore, each point in the SST dataset is categorized by the ecoregion in which it is located. This classification is based on the Marine Ecosystems Of the World (MEOW) developed and distributed by The Nature Conservancy. These data have been QA'd in that we have selected only data values with associated quality flags of 4-7. No data and bad data are given the NoData value = -9999.
    Purpose: The North Pacific nearshore temperature dataset was developed for the purpose of examining SST values and patterns recorded since 1981 at a moderate resolution. Moreover these data are offered in a format intended to facilitate their use between biogeographers and ecologists examining the effects of climate change in coastal regions.This metadata file is valid for each monthly mean SST shapefile for 09/1981 - 12/1981. Details given are for each individual file, not the comprehensive bundle of files for the year.
    Supplemental_Information: For access to the original HDF data, please visit the NASA Physical Oceanography Distributive Active Archive Center (PO.DAAC) at http://podaac.jpl.nasa.gov/

    Process_Step:
      Process_Description: Downloaded SST data in HDF from NODC and NASA PO.DAAC
      Source_Used_Citation_Abbreviation: AVHRR1
      Source_Used_Citation_Abbreviation: AVHRR2
    Process_Step:
      Process_Description: Manually renamed HDF files to make them suitable for import into ArcGIS.
    Process_Step:
      Process_Description: Used Marine Geospatial Ecology Tools (MGET) package within ArcGIS to convert SST and associated quality flag files in HDF into ArcGIS GRID files.
      Source_Used_Citation_Abbreviation: MGET
    Process_Step:
      Process_Description: Performed Map Algebra to scale SST data according to equation provided in data header.
      Source_Used_Citation_Abbreviation: AVHRR1
      Source_Used_Citation_Abbreviation: AVHRR2
    Process_Step:
      Process_Description: Used Spatial Analyst Extention on quality flag files to create a mask containing pixels with quality values of 4 and greater.
    Process_Step:
      Process_Description: Applied quality mask to each corresponding SST GRID using the ArcGIS Spatial Analyst "Extract by Mask" function.
    Process_Step:
      Process_Description: Used raster calculator to eliminate temperature values of < -2.0 degrees Celsius from the high-quality GRIDs.
    Process_Step:
      Process_Description: Created a buffer shapefile by using ArcToolbox Buffer tool to isolate a region from the coastline out to ~16 km (i.e. 4 pixel widths) of shoreline as defined by the Global Self-consistent, Hierarchical, High-resolution Shoreline.
      Source_Used_Citation_Abbreviation: GSHHS
    Process_Step:
      Process_Description: Clipped one high-quality raster with the Nearshore Buffer shapefile to yield a nearshore raster dataset.
    Process_Step:
      Process_Description: Used ArcGIS editor tools to manually clean up the nearshore point shapefile created in the previous step to constrain the file to about 4 pixels off of the shoreline as defined by GSHHS. This creates a generic nearshore point shapefile for use in extracting SST values from the global SST rasters.
      Source_Used_Citation_Abbreviation: GSHHS
    Process_Step:
      Process_Description: Used ArcGIS Spatial Analyst "Extract Values to Points" function to extract nearshore SST values from the centroids of the North Pacific SST rasters using the generic nearshore points shapefile created in the previous step.
    Process_Step:
      Process_Description: Used "Add XY Coordinates" ArcToolbox (Data Management Tools) tool to add latitude and longitude information to the Attributes table of each nearshore SST point shapefile.
    Process_Step:
      Process_Description: Used the ArcToolbox (Data Management Tools) "Add Join" tool to add a spatial join with The Nature Conservancy Marine Ecoregions of the World (MEOW) spatial data, giving each Nearshore SST point the attributes of the MEOW in which it is located.
      Source_Used_Citation_Abbreviation: MEOW
    Process_Step:
      Process_Description: Used R scripting language and the RODBC R package to transform shapefile attribute tables (.dbf) into .csv files (one .csv per table).
      Source_Used_Citation_Abbreviation: RODBC
